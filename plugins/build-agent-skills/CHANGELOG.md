# Changelog

All notable changes to the Agent Skills plugin will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-12-21

### Added
- Initial release of Agent Skills plugin
- `working-with-agent-skills` skill with comprehensive guidance
- Complete Agent Skills specification documentation
- YAML frontmatter schema reference (required and optional fields)
- Skill creation workflow from start to publish
- Validation instructions using `skills-ref` tool via `uvx`
- Cross-platform compatibility guidance for Claude Code, Cursor, GitHub Copilot
- Progressive disclosure strategy documentation
- Real-world examples: database migrations and AWS Lambda deployment
- Best practices for writing effective, discoverable skills
- Troubleshooting guide for common issues
- References to official resources (agentskills.io, GitHub repo)

### Documentation
- Comprehensive README.md with quick start guide
- LICENSE file (Apache 2.0)
- CHANGELOG.md (this file)

### Validation
- Skill validated against official Agent Skills specification
- Tested with `skills-ref validate` - passes all checks
- Cross-platform compatible structure
