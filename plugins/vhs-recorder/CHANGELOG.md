# Changelog

All notable changes to the VHS Recorder plugin will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-12-21

### Added

- Initial release of VHS Recorder plugin
- Comprehensive `vhs-recorder` skill with complete workflow guidance
- Detailed documentation covering:
  - VHS tape file structure
  - All VHS commands and settings
  - Timing strategies (Wait vs Sleep)
  - Hide/Show patterns for clean recordings
  - Typography and appearance presets
  - Common pitfalls and solutions
- Four example tape files:
  - `basic-demo.tape` - Simple introduction
  - `cli-tool-demo.tape` - CLI tool demonstration with hidden setup
  - `git-workflow.tape` - Multi-stage git workflow
  - `tutorial-with-errors.tape` - Realistic tutorial with corrections
- Quality checklist for validating recordings
- Complete plugin README with installation and usage instructions
- Examples directory with README explaining patterns

### Documentation

- Comprehensive SKILL.md with:
  - Step-by-step workflow
  - Complete command reference
  - Timing best practices (3-2-1 rule)
  - Configuration presets for different use cases
  - Advanced patterns and techniques
  - Testing and validation guidance
